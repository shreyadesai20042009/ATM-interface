A simple **ATM Interface project in Java** that simulates basic banking operations such as deposit, withdrawal, balance checking and account interaction through a console-based menu system.  
This project demonstrates core Object-Oriented Programming concepts in Java including classes, methods, user input, and control flow.

---

## 🧠 Project Overview

This ATM Interface project allows a user to:

- Create a bank account with initial balance  
- Deposit money  
- Withdraw money  
- Display current balance  
- Exit the system  

This is a **console-based banking simulation** built using Java and is suitable for beginners learning Java fundamentals. :contentReference[oaicite:0]{index=0}

## 🎯 How It Works

1. Program displays a simple menu to the user  
2. User selects one of the banking operations  
3. Input is taken via the Scanner class  
4. Balance is updated and displayed accordingly  
5. Loop continues until the user exits the interface
